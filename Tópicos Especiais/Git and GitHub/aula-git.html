Comandos Git

Configuração Inicial

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@provedor.com"

Primeira Vez

git clone https://github.com/seuusuariogit/repositorio.git

Checar Atualizações

git pull

Adicionar modicações

git add .

Empacotar Mudanças

git commit -m "Descreva as mudanças"

Ver Status do Repositório Local

git status

Enviar ao GitHub

git push
OBS: A primeira vez, vai solicitar que você faça o login no GitHub.
